# Python Microservice

A lightweight Python service demonstrating backend data processing capabilities.

## Overview

This service provides utility functions for data validation and processing that could be called by the main application via API endpoints.

## Requirements

- Python 3.8+
- pip

## Setup

```bash
cd services/python_service
pip install -r requirements.txt
```

## Usage

```bash
python service.py
```

## Features

- Data validation utilities
- JSON processing
- RESTful API endpoint example
