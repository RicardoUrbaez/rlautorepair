# Java Utility Service

A Java-based utility service for business logic and data processing operations.

## Overview

This service demonstrates enterprise-level Java capabilities including data structures, validation, and business rule processing.

## Requirements

- Java 11 or higher
- Maven 3.6+

## Build

```bash
cd services/java_service
mvn clean compile
```

## Run

```bash
mvn exec:java -Dexec.mainClass="com.rlautoshop.service.AppointmentProcessor"
```

## Features

- Appointment scheduling logic
- Customer data management utilities
- Business rule validation
- Data transformation services
